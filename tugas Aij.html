Soal Pilihan Ganda — Administrasi Infrastruktur Jaringan (AIJ)

Versi website: soal + kunci jawaban interaktif. (Sumber: Kisi-kisi Soal AIJ yang Anda unggah)

Catatan: beberapa soal bergambar saya sisakan placeholder — unggah gambar jika Anda ingin saya masukkan juga.
1. Berikut yang bukan syarat sebuah komputer jaringan lokal saling terhubung, yaitu ....
a. memiliki alamat host yang sama
b. memiliki jaringan alamat yang sama
c. memiliki alamat broadcast yang sama
d. menjalankan protokol jaringan yang sama
e. menggunakan sistem pengalamatan yang sama
Jawaban: a. memiliki alamat host yang sama
2. Jika perusahaan memiliki 300 komputer, mekanisme paling efektif adalah..
a. NAT
b. PAT
c. routing statis
d. routing dinamis
e. internet gateway
Jawaban: b. PAT
3. NAT yang berfungsi menyambungkan sebuah perangkat agar dapat diakses dari internet adalah....
a. basic NAT
b. static NAT
c. dynamic NAT
d. one to many NAT
e. many to many NAT
Jawaban: b. static NAT
4. Fitur yang mengubah akses koneksi dari jaringan publik ke jaringan lokal adalah....
a. Block Site
b. Redirect
c. srcnat
d. Dstnat
e. Jump
Jawaban: d. Dstnat
5. Fungsi dari Firewall yang tepat adalah...
a. Penghubung antara dua jaringan sekaligus
b. Mengatur dan mengontrol lalu lintas jaringan
c. Penghubung antara dua jaringan yang berbeda
d. Program untuk mendapatkan konten dari internet
e. Penghubung antara dua jaringan ke internet dengan satu IP
Jawaban: b. Mengatur dan mengontrol lalu lintas jaringan
6. Firewall NAT yang merubah alamat IP sumber (one-to-one) disebut ...
a. Src-nat
b. Redirect
c. Masquerade
d. Dst-nat
e. Reject
Jawaban: a. Src-nat
7. Firewall NAT yang merubah IP sumber saat jumlah client >1 adalah ...
a. Src-nat
b. Redirect
c. Masquerade
d. Dst-nat
e. Reject
Jawaban: c. Masquerade
8. Firewall NAT yang merubah alamat IP tujuan menjadi router itu sendiri disebut ...
a. Src-nat
b. Redirect
c. Masquerade
d. Dst-nat
e. Reject
Jawaban: d. Dst-nat
9. Action yang digunakan untuk menolak paket tanpa pemberitahuan disebut ...
a. Drop
b. Redirect
c. Logging
d. Accept view
e. Reject
Jawaban: a. Drop
10. Action yang menolak paket dengan pemberitahuan disebut ...
a. Drop
b. Redirect
c. Logging
d. Accept view
e. Reject
Jawaban: e. Reject
11. Firewall yang dapat kita gunakan untuk mencatat suatu aktivitas, misalnya ingin mencatat siapa saja yang melakukan ping ke router adalah...
a. Drop
b. Redirect
c. Logging
d. Accept view
e. Reject
Jawaban: c. Logging
12. Dibawah ini merupakan kelebihan GUI dibandingkan Text adalah...
a. Sulit digunakan
b. Konfigurasi sulit
c. User friendly
d. Tampilan kurang menarik
e. Semua benar
Jawaban: c. User friendly
13. Memiliki fungsi untuk mengubah destination address pada sebuah paket data (digunakan untuk membuat host lokal dapat diakses dari internet) disebut...
a. DSTNAT
b. DHCP NAT
c. FILTERING
d. SCNAT
e. REDIRECT
Jawaban: a. DSTNAT
14. (Soal bergambar) Gambar di bawah ini adalah Router.... (unggah gambar untuk jawaban pasti)
a. RB750gr2
b. RB750gr3
c. RB941Ui-2ND
d. RB940i
e. RB951Ui-2ND
Jawaban: (gambar tidak tersedia — tambahkan foto router untuk jawaban akurat)
15. Cara mengakses Router Mikrotik dibawah ini kecuali...
a. Komputer
b. Web Based
c. Telnet
d. SSH
e. Winbox
Jawaban (perkiraan): a. Komputer — opsi ini tampak tidak sesuai metode akses; jika maksud soal berbeda, unggah konteks/gambar.
16. Model arsitektural jaringan komputer yang dikembangkan sebagai standar komunikasi data adalah...
a. Barisan
b. Tahapan
c. ISO
d. OSI
e. IEEE
Jawaban: d. OSI
17. Berapakah jumlah host dari network address 192.168.1.64/27?
a. 30
b. 31
c. 33
d. 36
e. 39
Jawaban: a. 30 (prefix /27 → 32 alamat, 30 host usable)
18. Router beroperasi pada layer....
a. Data Link
b. Network
c. Data Center
d. Physical
e. Transport
Jawaban: b. Network
19. Pembagian sebuah network address menjadi kelompok-kelompok network lebih kecil biasa dikenal dengan istilah...
a. Subnet ID
b. Mac Address
c. Network
d. Netmask
e. Host ID
Jawaban: a. Subnet ID (istilah terkait subnetting)
20. Mengapa banyak pengguna mikrotik di Indonesia?
a. Harga Terjangkau dan Fitur Lengkap
b. Mudah didapatkan
c. Tampilannya terlihat bagus
d. Tidak mudah rusak
e. Sulit dicari
Jawaban: a. Harga Terjangkau dan Fitur Lengkap
21. Perbedaan mendasar antara Telnet dan SSH terletak pada aspek...
a. Tampilan
b. Keamanan
c. Kecepatan
d. Keindahan
e. Kemudahan
Jawaban: b. Keamanan (SSH terenkripsi; Telnet tidak)
22. Untuk instalasi Mikrotik RouterOS menggunakan netinstall, aplikasi yang dibutuhkan selain netinstall adalah...
a. RouterOS
b. Kabel LAN
c. Mikrotik
d. Linux
e. Routerboard
Jawaban: a. RouterOS
23. Sebelum melakukan proses remote (SSH/Telnet) yang perlu diketahui adalah...
a. DNS Server
b. Mail Server
c. IP Address
d. Mac Address
e. Subnet mask
Jawaban: c. IP Address
24. Alat yang berfungsi untuk menghubungkan dua jaringan dengan segmen yang berbeda adalah...
a. Switch
b. Modem
c. Access Point
d. Router
e. Hub
Jawaban: d. Router
25. Pada menu apakah jika kita ingin melakukan konfigurasi Identity pada router di mikrotik...
a. Firewall
b. Menu File > Syetem > Identity
c. New Terminal
d. System > Identity
e. System > Identitas
Jawaban: d. System > Identity
26. Berikut ini contoh media transmisi yang menggunakan kabel, kecuali...
a. Fiber Optic
b. SMTP
c. Coaxial / Coax / BNC
d. Wireless
e. STP
Jawaban: b. SMTP (bukan media, melainkan protokol)
27. Fungsi dari Winbox adalah sebuah software yang digunakan untuk meremote Mikrotik ke mode...
a. Terminal
b. OS
c. GUI (Graphical User Interface)
d. CLI
e. Semua Benar
Jawaban: c. GUI
28. Berikut ini kelebihan mikrotik adalah....
a. Banyak Fitur
b. Harga yang cukup Terjangkau
c. Dapat berperan sebagai Router
d. Mudah penggunaannya
e. Semua benar

29. Definisi untuk daerah yang dilayani oleh satu access point wireless LAN standard 802.11 a/b/g adalah...
